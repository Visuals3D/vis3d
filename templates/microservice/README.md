# ${MICROSERVICE_NAME} Microservice



A Microservice generated by the @visuals3d/vis3d cli tool.



## Run in development:



### With Docker

```bash
docker-compose up -d
```



### Without Docker

```bash
npm run watch:debug
```





## Run in production



### With Docker

```bash
docker build -t ${MICROSERVICE_NAME}
docker run -d -t ${MICROSERVICE_NAME} -e NODE_ENV=production
```



### Without Docker

```bash
npm run build
npm run prod:start
```
